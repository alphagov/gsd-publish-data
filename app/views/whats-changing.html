What's changing

How the changes will affect you.

1. New KPIs

There are several new KPIs being introduced. This is so that data users can understand more about the demand on different services across government.

Specific guidance on each KPI can be found in the Service Manual.

2. Monthly publishing rather than quarterly

This allows data users to see a more granular level of detail and understand trends better.

There is a consistent date for publishing every month, which is the last day of the following month. For example, data for the period 1 to 31 January must be published by 28 February.

3. New processes for publishing data

As before, you are responsible for making sure data is published for multiple services in your organisation.

However, we are making it easier to gather data. By uploading a list of all the contacts you currently ask for data, we can automate the process of sending out requests. Your contacts will then be able to upload data directly, rather than by emailing it back to you.

You also have the ability to monitor the progress of data uploads, so that you know who to chase up for missing data.
